SVGartiste

[ Incomplete ] My attempt at making an online tool that makes coding SVG slightly easier. The attributes of each element can be modified by number wheel, typing, or by clicking on the viewport. The order of layers can be adjusted by click and drag. Viewport can show customizable gridlines. When you are happy with the resulting SVG image, copy and paste the code to your project.
Minimap Comments
HTML
CSS
JavaScript
This simple app generates comments that are large enough to be legible on the minimap. The large comments generated by user text input is copied to the clipboard so the user can paste the string into the code. It can output comments for HTML, CSS and JavaScript. Best used with capital letters, but it can handle small-case letters and a few symbols.
