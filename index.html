SVGartiste
HTML
CSS
JavaScript
jQuery
[ Unfinished project ] My attempt at making an online tool that makes coding SVG slightly easier. The attributes of each element can be modified by number wheel, typing, or by clicking on the viewport. The order of layers can be adjusted by click and drag. Viewport can show customizable gridlines. When you are happy with the resulting SVG image, copy and paste the code to your project.
Large Comments
HTML
CSS
JavaScript
This tiny app takes text input from the user, generates large letters that takes up 6 lines of space, and stores the string in the clipboard for the user to paste into the script. It can output comments for HTML, CSS and JavaScript. Best used with capital letters, but it can handle small-case letters and a few symbols.
